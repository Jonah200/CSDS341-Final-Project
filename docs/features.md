# Use Cases
- Fans trying to do amateur analytics on their favorite teams
- Team analysts trying to identify skilled players
- Sports journalists identifying interesting trends

# Features
This database stands as a proof of concept, storing only a few basic statistics for players and basic pitch characteristics.

Fans can identify which players participated in a game

Analysts can identify the best players on each team by a range of metrics:
- Best batting average
- Best strikeout rate
- Fastest pitch by pitch type

Jornalists can idenify which players performed will in a game.
